## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/KhushalJangid/erp

# Access
$ cd erp

# Install dependencies
$ pip install -r requirements.txt

# Run the project
$ python manage.py runserver

# The server will initialize in the <http://localhost:3000>
```
